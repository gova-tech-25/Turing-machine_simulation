# 🧠 Turing Machine Simulator

An interactive, cyber-themed **Turing Machine simulator** that lets you visualize computation step-by-step. Design your own transition rules, watch the tape evolve in real time, and solve classic theoretical computer science problems.

---

## 🚀 Features

* 🎞️ **Real-time Tape Visualization**
  See how the tape changes as the machine reads, writes, and moves.

* 🔄 **Step-by-Step Execution**
  Play, pause, or step through each transition.

* 🧩 **Custom Rule Editor**
  Build and modify your own transition functions.

* 📊 **State Diagram Visualization**
  Automatically generated state graphs with active transitions highlighted.

* 📜 **Execution Log**
  Track every computation step with detailed logs.

* 🎯 **Built-in Challenges**

  * Palindrome Recognizer
  * aⁿbⁿ Recognizer
  * Binary Incrementer

* 🏆 **Gamified Badges System**
  Earn achievements by solving challenges.

---

## 🛠️ How to Use

1. Enter an input string on the tape.
2. Choose a preset or create your own rules in the **Program** tab.
3. Click **Play** or use **Step** to execute the machine.
4. Observe:

   * Tape updates
   * State transitions
   * Execution logs
5. To change the input, select the desired program, enter your new string, and press **Reset** to load it onto the tape.
